# Telegram Setup Notes

## What you need

- a Telegram bot token from BotFather
- the target chat id

## Chat id note

For direct chats, the chat id is usually the numeric user chat id.
For groups, it can be a negative id.

## Delivery strategy

Start in dry-run mode first.
Only enable real sends once the brief text is consistently useful.

## Recommendation

Use one private test chat first before wiring any scheduled sends.
